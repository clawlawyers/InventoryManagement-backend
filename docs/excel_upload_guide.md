# Excel Upload Guide for Salesmen

This guide explains how to format your Excel file for uploading client data.

## Required Excel Format

Your Excel file should have the following columns:

| Column Name | Description | Required |
|-------------|-------------|----------|
| name | Client's full name | Yes |
| phone | Client's phone number | Yes |
| email | Client's email address | No |
| address | Client's street address | Yes |
| city | Client's city | Yes |
| state | Client's state | Yes |
| zipCode | Client's ZIP/postal code | Yes |
| notes | Additional notes about the client | No |

## Example Excel File

Here's an example of how your Excel file should be formatted:

| name | phone | email | address | city | state | zipCode | notes |
|------|-------|-------|---------|------|-------|---------|-------|
| John Smith | 555-123-4567 | john@example.com | 123 Main St | Anytown | CA | 12345 | Prefers calls in the morning |
| Jane Doe | 555-987-6543 | jane@example.com | 456 Oak Ave | Somewhere | NY | 67890 | New client |

## Upload Instructions

1. Create your Excel file following the format above
2. Save the file as .xlsx or .xls format
3. Log in to your salesman account
4. Navigate to the "Upload Clients" section
5. Click "Choose File" and select your Excel file
6. Click "Upload"
7. Review the upload results

## Notes

- Each phone number must be unique in the system
- All required fields must be filled in for each client
- Maximum file size is 5MB
- Only Excel files (.xlsx or .xls) are accepted

If you encounter any issues with your upload, please check the error messages and correct your Excel file accordingly.
